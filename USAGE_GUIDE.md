# 📖 功能使用指南

## 激活码管理

### 生成激活码

#### 单个生成
1. 进入管理后台 → 激活码管理
2. 点击 "生成激活码" 按钮
3. 填写码的备注信息
4. 选择 VIP 码或普通码
5. 确认生成

#### 批量生成
1. 点击 "批量生成" 按钮
2. 输入数量（最多1000个）
3. 自动生成随机激活码

### 激活码状态

| 状态 | 说明 |
|------|------|
| **未使用 (unused)** | 可被任何用户使用 |
| **已使用 (used)** | 已被激活，显示使用者信息 |
| **已过期 (expired)** | 超出有效期，不可再使用 |

### 激活码操作

- 查看激活码详情
- 删除过期激活码
- 修改激活码备注

## 用户管理

### 用户同步

- 自动从微信后台获取粉丝列表
- 记录用户订阅时间、昵称、头像等信息
- 支持手动刷新和定时同步

### 用户激活状态

| 状态 | 说明 |
|------|------|
| **未激活** | 用户未使用激活码 |
| **待激活** | 用户已获得激活码，需验证 |
| **已激活** | 用户已验证激活码，获得全部权限 |

### 用户操作

- 查看用户详情（地区、最后登录等）
- 搜索和筛选用户
- 批量激活或管理用户
- 查看用户使用的激活码

## 自定义回复 ⭐

### 文本回复

```json
{
  "keyword": "hello",
  "reply_type": "text",
  "content": "你好，欢迎使用本服务！",
  "match_type": "exact",
  "priority": 100,
  "enabled": true
}
```

### 匹配模式

- **精确匹配 (exact)** - 用户消息与关键词完全相同
- **包含匹配 (contains)** - 用户消息包含关键词

### 优先级控制

- 多个关键词时，按优先级高低排序
- 默认优先级为50
- 优先级越高，越先被匹配

## 触发关键词

### 配置激活码关键词

1. 进入 "触发关键词" 页面
2. 输入关键词（如："激活", "激活码"）
3. 配置回复模板
4. 设置优先级
5. 启用后生效

### 使用流程

1. 用户在微信公众号回复关键词
2. 系统自动触发激活码生成
3. 向用户发送激活码
4. 用户验证激活码激活
